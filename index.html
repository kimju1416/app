📖 Easy App Builder V3.1 완전 활용 가이드

1

AI 대상을 먼저 선택하세요

헤더의 Claude / GPT / Gemini 버튼으로 사용할 AI를 선택하면, 해당 AI에 최적화된 지시어가 프롬프트에 자동 포함됩니다.
Claude: XML 구조, 복잡한 로직에 강함 ✅ 추천
ChatGPT: Markdown 구조, 빠른 응답
Gemini: 구글 생태계(시트/드라이브) 최적화

2

주제 입력 후 기능 상세 내용이 핵심!

칩 클릭 + 직접 규칙 작성을 함께 쓰세요.
예: "A열=학번, B열=이름", "1인 1회 신청 제한", "퀴즈 랜덤 10문제"
→ 구체적일수록 AI 코드 품질이 확실히 올라갑니다!

3

두 가지 방법으로 코드 만들기

📋 프롬프트 복사: 클립보드 복사 → ChatGPT/Claude/Gemini 창에 붙여넣기(Ctrl+V)
🤖 AI로 코드 생성: 버튼 클릭 → 이 앱 안에서 Claude가 즉시 Code.gs + Index.html 생성!

4

GAS 에디터에 붙여넣고 배포!

① script.google.com → 새 프로젝트
② Code.gs 내용 붙여넣기
③ [+] 버튼 → Index.html 파일 생성 → 내용 붙여넣기
④ [배포] → [새 배포] → 웹 앱 → 액세스: 모든 사용자 → 배포 → URL 공유!

개발자 꿀팁 (Tip & Tricks)

  • 💬 한 번에 완벽할 수 없어요: 코드 실행 후 "버튼 색 바꿔줘", "모바일에서 글씨가 너무 작아" 식으로 AI에게 계속 수정 요청하세요. 티키타카 5번이면 대부분 완성!
  • 📊 구글 시트 연동 팁: 기능란에 "A열=학번, B열=이름..." 처럼 열 구조를 명시하면 AI가 딱 맞는 코드를 작성합니다.
  • ⚡ Claude 추천 상황: 복잡한 조건 로직, 다단계 검증, 이메일 자동 발송이 포함된 앱은 Claude가 가장 깔끔하게 짜줍니다.
  • 🔧 오류 발생 시: 오류 메시지를 그대로 AI에게 붙여넣으면 즉시 수정해 줍니다. "이 오류가 났어요: [오류메시지]" 라고만 해도 충분!
  • 🚀 빠른 배포 팁: script.google.com → Apps Script → 새 프로젝트 → 이름 지정 → Code.gs + Index.html 붙여넣기 → 배포
